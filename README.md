# 人事管理系统 java1041

## 项目概述

人事管理系统是一款基于Java和SpringBoot框架开发，结合VUE实现前后端分离的B/S架构应用。该系统旨在提高企业人事管理的效率和准确性，涵盖员工管理、考勤、工资、培训等多个模块。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**: 
  - IDEA
  - JDK1.8
  - Maven
  - Mysql 5.7+
- **技术栈**: 
  - Java
  - Mysql
  - SSM（Spring、SpringMVC、MyBatis）
  - Mybatis-Plus
  - VUE
  - jQuery
  - HTML
- **数据库可视化工具**: Navicat
- **服务器**: SpringBoot内嵌Apache Tomcat
